# Brands

## 1. Get all brands

```
[GET] api/brands
```

### Example

#### GET

```
Link: api/brands
```

#### Response

```
Status: 200 OK
```

```json
{
  "data": [
    {
      "id": 1,
      "name": "LG",
      "imageUrl": "/static/images/logo.png",
      "link" : "https://www.lg.com",
    }
  ]
}
```
